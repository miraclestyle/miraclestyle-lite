Seller rules impose business policies and regulations on commercial transactions. Seller rules operate as part of the checkout process logic within shopping cart. System depends on some seller rules to be in place for normal operation. All active seller rules are executed sequentially from top to bottom.
Seller rules will appear in the order you have manually ordered them.
Click "Add" action to add new rule.
Click on a rule to edit rule.
Use drag handle to reorder a rule by dragging it within its group.
Use drag handle to remove a rule by dragging it outside the left edge of the screen.
Address
Address rule controls billing or shipping address that buyer submits during shopping cart checkout. System requires that seller has exactly two active address rules, one for billing and the other for shipping address.
Carrier
Carrier rule defines delivery method that will be matched and calculated against order, and become available as an option to buyer during shopping cart checkout. System requires that seller has at least one active carrier rule. System supports multiple active carrier rules.
Currency
Currency rule establishes accepted currency. This rule makes an exception in terms of operation as it operates on both, catalog product unit prices, and checkout process logic within shopping cart. System requires that seller has exactly one active currency rule.
Discount
Discount rule defines product discount that will be matched, calculated, and applied against order lines during shopping cart checkout. System does not require that seller has active discount rules. System supports multiple active discount rules.
PayPal
PayPal rule defines PayPal as acceptable payment method that buyer can use to make a payment during shopping cart checkout. System does not require that seller has active PayPal rules. System supports one active PayPal rule.
Tax
Tax rule defines sales tax that will be matched, calculated, and applied against order lines during shopping cart checkout. System does not require that seller has active tax rules. System supports multiple active tax rules.
