As their name implies, seller rules are principles that impose business policies on customers. Most seller rules operate as part of the logic of the checkout process within shopping cart. System requires at least one active and configured seller rule of each kind in order to operate correctly. All active seller rules are executed sequentially from top to bottom.
Your seller rules will appear in the order you have manually ordered them.
Click "Add" action to add new rule.
Click on a rule to edit rule.
Use drag handle to reorder a rule by dragging it within its group.
Use drag handle to remove a rule by dragging it outside the left edge of the screen.
Address
Use address rule to define billing and shipping address restrictions during purchase. System requires exactly two address rules, one for billing and other for shipping address.
Carrier
Use carrier rule to define available delivery method during purchase. One carrier rule coresponds to one delivery method. Each carrier rule can be configured to dynamically calculate shipping and handling prices based given circumstances. System requires at least one carrier rule. You can define multiple carrier rules, and all applicable carrier rules will be available to a customer during purchase.
Currency
Use currency rule to define a currency you accept. System requires exactly one currency rule.
Discount
Use discount rule to define discount given under desired circumstances during purchase. System does not require discount rules. You can define multiple discount rules, and all discount rules that match during purchase will be applied accordingly.
PayPal
Use PayPal rule to define PayPal as acceptable payment method during purchase. System does not require PayPal rules. You can define one PayPal rule.
Tax
Use tax rule to define taxes during purchase. System does not require tax rules. You can define multiple tax rules, and all tax rules that match during purchase will be applied accordingly.
