Use product variant configurations to override default product values for specific product customization during purchase. Product variant configurations can override any default product value except product category and name. When customer customizes the product by selecting desired variant options on the product view form, product values will be altered according to the first product variant configuration that matches the customized product.
Product variant configurations will appear ordered the way you have manually ordered them.
Click the "Add" button to add a new variant configuration.
Click on a variant configuration to edit it.
Use reorder to reorder a variant configuration by dragging it within its group.
Use reorder to remove a variant configuration by dragging it outside the left edge of the screen.
